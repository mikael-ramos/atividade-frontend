HTML (HyperText Markup Language) é uma linguagem de marcação usada para criar e estruturar páginas web. ele pode ser definido como o "esqueleto" ou a base de qualquer site,definindo os elementos como textos, imagens, links e formulários. Alem disso ele serve para estruturar o conteúdo como organizar títulos, parágrafos, listas, tabelas, etc. O HTML não é uma linguagem de programação ele não tem lógica como loops ou condições.

CSS (Cascading Style Sheets, ou "Folhas de Estilo em Cascata") é uma linguagem de estilo usada para controlar a aparência de elementos em uma página web Ele é como a "maquiagem" ou o "designer" da web: define cores, fontes, layouts, animações e muito mais, serve para estilizar o HTML transformando os elementos HTML brutos em algo visualmente atraente, servindo como meio de separação de conteúdo facilitando a manutenção da pagina e para criar consistência visual, definindo um padrão de cores, fontes e tamanhos para todo o site.
